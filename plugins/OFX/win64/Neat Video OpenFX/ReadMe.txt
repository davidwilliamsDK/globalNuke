1. INTRODUCTION

Neat Video is a filter plug-in designed to reduce visible noise
in digital video sequences.

The key features of the Neat Video OpenFX plug-in:

  - Both intra-frame and temporal filtration
  - Advanced Noise Reduction to reduce noise and grain in video sequences
  - Smart Sharpening to make video look sharper without amplification of noise
  - Automatic noise analyzer to adapt the plug-in to any video sources
  - High-bitdepth rendering (in Pro plug-in)
  - CUDA and OpenCL acceleration to speed up processing using computation-capable GPUs


Neat Video OpenFX plug-in (Win) is produced in two editions - Demo and Pro:
- Demo plug-in is a freeware edition of the software with limited functionality
- Pro plug-in supports frames of any size; regular and high-bitdepth rendering


2. SYSTEM REQUIREMENTS
 - An OpenFX-compatible host application 
   - Foundry Nuke 7.0+ / 6.0+ / 5.1+
   - Assimilate Scratch 7.0+ / 6.1+
   - Eyeon Fusion 6.0+ / 5.3+
   - HS-ART DustBuster+ 5.0+ / 4.5+
 - Windows 8, Windows 7, Vista, XP (32-bit or 64-bit)
 - Pentium-III class machine or better
 - 1 GB RAM or higher
 - True color display with 1024x768 resolution or higher
