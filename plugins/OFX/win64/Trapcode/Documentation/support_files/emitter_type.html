Emitter tab> Emitter Type pop-up

Particles are born and released from the Emitter tab's Emitter Type pop-up. The Emitter Type is very important since it sets the shape of the particle emission. Animated examples of each Emitter Type are shown below.

 

The Emitter Type options. Point is the default.

 

 

Seven Emitter Types

There are seven Emitter Types. Each activates different parameters in the Emitter tab as well as other parameters. The Emitter tab's Layer Emitter group, Grid Emitter group and Multi Emitter group are not active until the Emitter Type is changed to a Layer, Grid or Multi Emitter.

 

Point
All particles are born at the same point. This is the default Emitter Type.

Box
Particles are born within a box. For sake of illustration, Velocity is 0 here.

Sphere
Particles are born within a sphere. For sake of illustration, Velocity is 0 here.

Grid
Particles are born on a 2D or 3D grid. Read more about Grid Emitter.

Multi Emitter(s)
Particles are born from multiple emitters. Read more about Multi Emitters.

Layer
Particles are born from an image node in the comp. Read more about Layer Emitters.

Layer Grid
Particles are born from an image node in the comp in the shape of the Grid Emitter. Read more about Layer Grid.
